## 🚀 О проекте
**«Своя База»** — это независимая легковесная СУБД, написанная на объектно-ориентированном C++ с соблюдением строгих архитектурных ограничений. 

В отличие от монолитных приложений, вся логика обработки SQL-запросов, парсинга, управления транзакциями памяти и дискового ввода-вывода вынесена в **отдельную C++ библиотеку**. Клиентский код выполняет роль тонкого интерфейса (CLI), взаимодействуя с данными исключительно через публичные методы абстракции, что полностью исключает прямое манипулирование файлами или внутренними структурами таблиц.

## ✨ Возможности (MVP)
- [x] **Настоящая персистентность:** Данные полностью сохраняются между перезапусками системы. При повторном запуске и выборе БД все таблицы и схемы мгновенно восстанавливаются.
- [x] **Архитектурная изоляция:** Строгое разделение на ядро (`libmysqldb.a`) и клиентское приложение (`MyDatabaseSystem`).
- [x] **Множественный INSERT:** Возможность пакетной вставки данных одной командой (`VALUES (1, 'A'), (2, 'B')`).
- [x] **Парсер на регулярных выражениях:** Полная поддержка БНФ-грамматики (Форма Бэкуса — Наура) без внешних зависимостей.
- [x] **CRUD-фильтрация:** Полноценная поддержка условных операторов `WHERE` (`=`, `!=`, `<`, `>`, `<=`, `>=`) для селективных выборок, точечных обновлений и удалений.

## 🏗 Архитектура и паттерны проектирования

В соответствии с требованиями ТЗ, ядром архитектуры системы является структурный паттерн **Фасад (Facade)**.

### Реализация паттерна:
Центральный класс `DatabaseFacade` (файлы `include/Database.h` и `src/Database.cpp`) объединяет разрозненные внутренние интерфейсы низкоуровневых сущностей в единое высокоуровневое окно управления.
